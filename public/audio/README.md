# Add Your Music Here

This directory is where the application looks for music files.

Please add three MP3 files to this directory and name them:

1.  `song1.mp3`
2.  `song2.mp3`
3.  `song3.mp3`

The application is pre-configured to load these specific files. If you want to use different file names, you can update them in `src/components/luminaudio-dashboard.tsx`.
